Réalisation d'un bugtracker

Travail individuel

Sujet : Dans le cadre d'une situation d'apprentissage évaluée, j'ai confectionné un bugtracker. Ce projet nous permettait de relier nos connaissances sur deux choses : le langage PHP et les bases de données.

Langages utilisés :

  • PHP
  • MySql
  • HTML / CSS

Ce que j'en retiens : Ce projet m'a permis de consolider mes compétences en PHP et en base de données. J'ai surtout découvert comment lier les deux afin de produire du contenu.

Réalisation de l'identité graphique d'un skatepark fictif

Travail individuel

Sujet : Dans le cadre d'une ressource intitulée culture artistique, j'ai élaboré l’identité d’un skate-park dont le nom est Spine Skatepark. Pour cela, il fallait réaliser un logo, une iconographie et une maquette de site "one page".

Logiciels utilisés :

  • Photoshop
  • Illustrator

Ce que j'en retiens : En définitive, cela m'a permis de consolider mes compétences sur des logiciels et de réfléchir à la cohérence entre chaque livrable. De plus, j'ai appris à réaliser un énoncé de démarche afin de justifier mes choix.

Réalisation d'un site de foodtruck fictif

Travail individuel

Sujet : Dans le cadre d'une situation d'apprentissage évaluée, j'ai élaboré un site web d'un foodtruck fictif. J'ai décidé de le faire sur le thème de la cuisine japonaise moderne.

Langages utilisés :

  • HTML
  • CSS
  • JavaScript

Ce que j'en retiens : Il s'agit de ma première réalisation de site web complet, fait au début de ma formation. Cela m'a montré l'ampleur d'un projet comme celui-ci, ainsi que des compétences que cela nécessite.

Réalisation d'une affiche liée au BUT MMI

Travail individuel

Sujet : Le but était de faire une affiche qui montre la multiplicité des compétences spécifiques à MMI. Pour cela, j'ai réalisé une affiche qui montre les différentes casquettes qu'ont les étudiants en MMI.

Outils utilisés :

  • Photoshop
  • Adobe Color

Ce que j'en retiens : J'ai pu augmenter mes capacités en graphisme et cela a développé ma créativité.
