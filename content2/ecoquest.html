Travail en groupe

Sujet : Dans le cadre d'une situation d'apprentissage évaluée, nous avons élaboré un site web qui vise à sensibiliser les gens à propos de leur impact sur l'environnement. Lors de ce projet, je me suis occupé du développement back avec la gestion de la base de données et du bon fonctionnement des fonctionnalités proposées.

Langages utilisés :

  • PHP
  • MySQL
  • HTML / CSS

Ce que j'en retiens : Ce projet m'a permis de consolider mes compétences en PHP et en base de données. J'ai égamement vu les difficultés possibles lièes au travail de groupe ou de la gestion de projet.

Le résultat final : Site web Ecoquest
